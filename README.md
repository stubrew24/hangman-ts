# Hangman

Hangman game built using Typescript

## Installation

### Clone

- Clone this repo to your local machine using `https://github.com/stubrew24/hangman-ts`

### Setup

```shell
$ npm install
$ npm start
```

## Built With

- [Typescript](https://www.typescriptlang.org/)
- [Webpack](https://webpack.js.org/)
